# 后台管理系统开发规划

## 一、用户管理

### 1.1 会员管理
**页面路径：** `/admin/users/members`

**搜索字段：**
- 用户名（输入框）
- 手机号（输入框）
- 邮箱（输入框）
- 注册时间（日期范围选择）
- 会员状态（下拉选择：全部/有效/已过期）

**列表字段：**
- 用户ID
- 用户名
- 手机号
- 邮箱
- 注册时间
- 会员卡类型
- 会员卡到期时间
- 会员状态
- 操作

**列表操作：**
- 查看详情
- 编辑用户信息
- 重置密码
- 禁用/启用账号
- 删除用户

**新增/编辑表单字段：**
- 用户名（必填，字符串，3-20位，字母数字下划线）
- 手机号（必填，字符串，11位，1开头）
- 邮箱（选填，字符串，邮箱格式验证）
- 密码（新增必填，编辑选填，字符串，6-20位）
- 角色（必填，下拉选择：member/coach/admin）

## 二、课程管理

### 2.1 课程列表
**页面路径：** `/admin/courses`

**搜索字段：**
- 课程名称（输入框）
- 教练（下拉选择，关联users表role=coach）
- 开课时间（日期范围选择）
- 课程状态（下拉选择：全部/未开始/进行中/已结束）

**列表字段：**
- 课程ID
- 课程名称
- 教练姓名
- 开课时间
- 已预约人数
- 课程容量
- 课程状态
- 操作

**列表操作：**
- 查看详情
- 编辑课程
- 删除课程
- 查看预约列表
- 查看签到记录

**新增/编辑表单字段：**
- 课程名称（必填，字符串，1-255位）
- 教练（必填，下拉选择，关联users表instructor_id）
- 开课时间（必填，日期时间选择）
- 课程容量（必填，数字，1-100）
- 课程描述（选填，文本域，最多1000字）
- 课程状态（选填，下拉选择：未开始/进行中/已结束）
